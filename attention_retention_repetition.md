# Complete Protocol

In your model:

```javascript
function processSignal(signal){

    attention = calculatePriority(signal);

    if(attention < threshold){
        return "ignored";
    }

    mammalianScore =
        emotionalValue(signal);

    reptilianScore =
        survivalValue(signal);

    cortexScore =
        logicalValue(signal);

    totalValue =
        reptilianScore * 0.6 +
        mammalianScore * 0.3 +
        cortexScore * 0.1;

    if(totalValue > acceptanceThreshold){

        allocateAttention(signal);

        if(rewardReceived(signal)){

            increaseRetention(signal);

            if(repeatedEnough(signal)){

                createHabit(signal);

                if(identityAttached(signal)){

                    becomePartOfSelf(signal);

                }
            }
        }
    }
}
```
