For a **neuroscience learning platform**, the UI should communicate:

* **calmness**
* **clarity**
* **scientific credibility**
* **focus**

Avoid aggressive colors (red/orange). Use **soft, cognitive-friendly colors**.

---

# 1. Recommended Color Palette

Use only **3 main colors** consistently.

| Color             | Use        | HEX       |
| ----------------- | ---------- | --------- |
| Deep Brain Blue   | primary    | `#2C3E50` |
| Calm Teal         | secondary  | `#3AAFA9` |
| Soft Neural White | background | `#F4F7F9` |

Optional accent (very limited):

```
#A8DADC (soft neural highlight)
```

Why this works:

* **blue → trust + intelligence**
* **teal → calm + neuroscience feel**
* **white → clarity + reading comfort**
