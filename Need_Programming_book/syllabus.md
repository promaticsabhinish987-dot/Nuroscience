# Ultimate Compression

If you compress the entire "Human Programming Language" into a few core variables:

```java
survival
belonging
status
curiosity
mastery
meaning
```

and a few core resources:

```java
attention
time
energy
```

Then almost every human behavior can be modeled as:

```java
HumanAction =
    optimize(
        survival,
        belonging,
        status,
        curiosity,
        mastery,
        meaning
    )
subjectTo(
        time,
        attention,
        energy
    );
```

That is the closest thing to a "human programming language" from psychology, behavioral economics, neuroscience, and evolutionary biology. The deeper you study it, the more you discover probabilities and tendencies—not absolute control. Humans are adaptive systems, not deterministic code.
